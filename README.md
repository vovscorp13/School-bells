# School-bells
Запуск звонков или любой мелодии по расписанию на неделю.
Просто в шаблоне расписания создаете свои значения и выбираете файл.
Затем в дни недели создаете.
Есть возможность выбора дня и извенения на тот день, не трогая шаблон.
Не забывайте сохранять изменения и выбор дня он не работает как текущий.
